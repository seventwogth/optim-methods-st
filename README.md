# Методы оптимизации

В репозитории собраны учебные реализации методов одномерной оптимизации для лабораторных работ по курсу методов оптимизации.

## Структура проекта

- `second/` — скрипты из предыдущего задания для функции `f(x)=sin(5x)·cos(0.5x)`.
- `third/` — решение лабораторной работы №3 для **варианта 16** методом квадратичной аппроксимации.
- `docs/` — исходные материалы задания и лекций.

## Лабораторная работа №3: квадратичная аппроксимация

Для варианта 16 по условию из `docs/lab_03_zadanie.pdf` используется функция

\[
f(x)=x^2-3x+x\ln x,\qquad [a,b]=[1,2],\qquad \varepsilon=10^{-4}.
\]

Важно: в целевой функции стоит именно произведение `x ln x`, а не дробь `x / ln x`.

### Идея метода

Метод квадратичной аппроксимации заменяет исходную функцию на параболу, построенную по трём точкам, и в качестве следующего приближения берёт вершину этой параболы.

1. Берутся три точки `x1 < x2 < x3`.
2. Вычисляются значения `f(x1)`, `f(x2)`, `f(x3)`.
3. Через эти точки строится интерполяционный квадратный многочлен.
4. Его вершина `u` считается новым приближением к экстремуму.
5. Из набора точек выбирается новая тройка, которая сохраняет область поиска около наилучшего значения функции.

Если обозначить `f1=f(x1)`, `f2=f(x2)`, `f3=f(x3)`, то абсцисса вершины параболы вычисляется по формуле

\[
u=x_2-\frac{(x_2-x_1)^2(f_2-f_3)-(x_2-x_3)^2(f_2-f_1)}{2\left((x_2-x_1)(f_2-f_3)-(x_2-x_3)(f_2-f_1)\right)}.
\]

### Инициализация по условию

В этом задании начальная точка берётся как середина интервала, а шаг `\Delta` — как четверть длины интервала:

\[
x^{(0)}=\frac{a+b}{2}=1.5,
\qquad
\Delta=\frac{b-a}{4}=0.25.
\]

Поэтому стартовая тройка имеет вид

\[
x_1=x^{(0)}-\Delta=1.25,
\qquad
x_2=x^{(0)}=1.5,
\qquad
x_3=x^{(0)}+\Delta=1.75.
\]

### Критерий останова

В программе используется критерий останова из задания:

\[
|x_{k+1}-x_k|<\varepsilon,
\qquad
\varepsilon=10^{-4}.
\]

### Что получается для варианта 16

Для данной функции

\[
f'(x)=2x-2+\ln x.
\]

На всём отрезке `[1,2]` выполнено `f'(x) \ge 0`, причём `f'(1)=0`, поэтому функция на отрезке не убывает и минимум достигается на левой границе:

\[
x_{\min}=1,
\qquad
f(x_{\min})=-2.
\]

То есть метод быстро приводит к граничной точке минимума.

## Содержимое `third/`

- `third/quadratic_approximation.py` — минималистичная реализация метода квадратичной аппроксимации, которая печатает последовательность приближений, найденный минимум и число итераций.
- `third/plot_minimum.py` — отдельный скрипт на `matplotlib`, который строит график функции на `[1,2]`, отмечает точку минимума и сохраняет `third/minimum.png`.

PNG-файл **не хранится в репозитории**: он создаётся пользователем при запуске скрипта.

Для `third/plot_minimum.py` требуется установленный пакет `matplotlib`.

## Запуск

```bash
python third/quadratic_approximation.py
python third/plot_minimum.py
```

После запуска второго скрипта появится файл:

```text
third/minimum.png
```
