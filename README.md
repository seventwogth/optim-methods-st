# Методы оптимизации для `f(x) = sin(5x) * cos(0.5x)`

Этот репозиторий содержит учебные скрипты для поиска экстремумов функции

\[
f(x)=\sin(5x)\cdot\cos(0.5x)
\]

на отрезке `[-4, 4]` с точностью `1e-8`.

## Структура проекта

В `second/` находится ровно 9 скриптов:

- `dichotomy_min.py` / `dichotomy_max.py` — метод дихотомии (половинного деления) для минимума/максимума.
- `golden_section_min.py` / `golden_section_max.py` — метод золотого сечения для минимума/максимума.
- `secant_min.py` / `secant_max.py` — метод хорд для уравнения `f'(x)=0`.
- `newton_min.py` / `newton_max.py` — метод Ньютона для уравнения `f'(x)=0`.
- `plot_extrema.py` — строит график с двумя видимыми экстремумами и сохраняет SVG.

## Обзор алгоритмов

### 1. Метод дихотомии

Для локального унимодального интервала `[a, b]`:

1. Вычисляются точки около середины:
   - `x1 = (a + b - delta) / 2`
   - `x2 = (a + b + delta) / 2`
2. Для минимума:
   - если `f(x1) <= f(x2)`, оставляем левую часть (`b = x2`), иначе правую (`a = x1`).
3. Для максимума сравнение инвертируется.
4. Остановка при `(b-a)/2 <= eps`.

### 2. Метод золотого сечения

Тоже требует унимодальный интервал, но на каждом шаге переиспользует одно из уже вычисленных значений функции.

- `tau = (sqrt(5)-1)/2`
- `x1 = b - tau*(b-a)`, `x2 = a + tau*(b-a)`
- Интервал сужается за счёт отбрасывания «худшей» стороны.

### 3. Метод хорд (`f'(x)=0`)

Экстремум рассматривается как стационарная точка (корень производной):

\[
x_{k+1}=x_k-f'(x_k)\frac{x_k-x_{k-1}}{f'(x_k)-f'(x_{k-1})}
\]

Остановка при выполнении двух условий:

- `|x_{k+1}-x_k| <= eps`
- `|f'(x_{k+1})| <= eps`

### 4. Метод Ньютона (`f'(x)=0`)

Используются первая и вторая производные:

\[
x_{k+1}=x_k-\frac{f'(x_k)}{f''(x_k)}
\]

Критерии остановки:

- `|x_{k+1}-x_k| <= eps`
- `|f'(x_{k+1})| <= eps`

## Стратегия инициализации

Каждый скрипт выполняет плотное сканирование на `[-4, 4]`, чтобы подобрать хорошую стартовую точку/интервал возле нужного глобального экстремума. Это делает все скрипты автономными и более устойчивыми.

## Построение графика

`second/plot_extrema.py` строит график функции на `[-0.8, 0.8]`, где видны только два экстремума, и сохраняет файл:

- `second/extrema_plot.svg`

## Запуск

```bash
python second/dichotomy_min.py
python second/dichotomy_max.py
python second/golden_section_min.py
python second/golden_section_max.py
python second/secant_min.py
python second/secant_max.py
python second/newton_min.py
python second/newton_max.py
python second/plot_extrema.py
```

Каждый скрипт метода выводит точку экстремума, значение функции и число итераций.
